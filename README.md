# Portfolio Website

This is a static portfolio website for Lois Chiles Concepcion, built for deployment on Vercel.

## Files

- `index.html` — main portfolio page
- `styles.css` — responsive styling
- `script.js` — simple navigation toggle behavior

## Deploying to Vercel

1. Create a GitHub repository and push this project.
2. Go to https://vercel.com and connect your GitHub account.
3. Import the repository and deploy the project.

Vercel will automatically detect a static site and deploy `index.html`.

## Notes

- The site includes sections for About, Projects, Skills, Experience, Education, and Contact.
- Projects are linked to GitHub repositories for easy review.
